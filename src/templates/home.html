Lancer une campagne en quelques étapes simples

Voici un guide très simple pour créer et envoyer votre première campagne de sensibilisation :

  1. Écrivez le mail que vous souhaitez envoyer dans Modèles d'email.
  2. Créez une fausse page (facultatif) vers laquelle vos contacts seront redirigés, dans la section Landing pages.
  3. Ajoutez un groupe de personnes que vous souhaitez sensibiliser dans Groupes.
  4. Indiquez l’adresse que vous utiliserez pour envoyer vos e-mails dans Profil SMTP.
  5. Une fois tout prêt, allez sur Nouvelle campagne pour lancer votre campagne.
  6. Vous pouvez ensuite suivre les résultats dans Suivi des campagnes.
  7. (Optionnel) Si vous souhaitez que le mail soit généré automatiquement, vous pouvez configurer l’intelligence artificielle dans Paramètres du LLM.
