Pour lancer l'application:
Depuis le dossier racine du projet, lancer la commande suivante:

python app.py

Bien penser a Ctrl+C pour arreter le serveur Flask et le docker proprement.

Ca va vous lancer automatiquement le docker et le serveur Flask. Tout est sensé fonctionner correctement.

https://127.0.0.1:3333/ pour accéder à l'interface de GoPhish (compte: admin, mot de passe: Voir le discord)
http://127.0.0.1:5000/ pour accéder à l'interface de SmartPhish
